Tar

Unix和类Unix系统上的压缩打包工具，最初的设计目的是将文件备份到磁带上（tape archive），因而得名tar。

tar 功能 选项 文件

功能

  • -c，--create 创建新的tar文件
  • -x，--extract，--get 解开tar文件
  • -t，--list 列出tar文件中包含的文件的信息
  • -r，--append 附加新的文件到tar文件中
  • -u，--update 用已打包的文件的较新版本更新tar文件
  • -A，--catenate，--concatenate 将tar文件作为一个整体追加到另一个tar文件中
  • -d，--diff，--compare 将文件系统里的文件和tar文件里的文件进行比较
  • --delete 删除tar文件里的文件。注意，这个功能不能用于已保存在磁带上的tar文件！

选项

  • -v，--verbose 列出每一步处理涉及的文件的信息，只用一个v时，仅列出文件名，使用两个v时，列出权限、所有者、大小、时间、文件名等信息。
  • -k，--keep-old-files 不覆盖文件系统上已有的文件
  • -f，--file [主机名:]文件名 指定要处理的文件名。可以用-代表标准输出或标准输入。f是开关参数中最后必须参数，用于指定档案名。
  • -P，--absolute-names 使用绝对路径，不推荐使用。
  • -j，--bzip2 调用bzip2执行压缩或解压缩。注意，由于部分老版本的tar使用-I实现本功能，因此，编写脚本时，最好使用--bzip2。
  • -J，--xz，--lzma 调用XZ Utils执行压缩或解压缩。依赖XZ Utils。
  • -z，--gzip，--gunzip，--ungzip 调用gzip执行压缩或解压缩
  • -Z，--compress，--uncompress 调用compress执行压缩或解压缩

缩写

tar代表未压缩的tar文件包。已压缩的tar文件则附加压缩文件的扩展名，如经过gzip压缩后的tar文件，扩展名为“.tar.gz”。
由于受到DOS8.3文件名格式的限制，常使用下列缩写：

  • .tgz等价于.tar.gz
  • .tbz与tb2等价于.tar.bz2
  • .taz等价于.tar.Z
  • .tlz等价于.tar.lzma
  • .txz等价于.tar.xz

操作

  • 压缩

    • tar –cvf archive.tar *.txt // 将目录里所有txt文件打包成archive.tar包
    • tar –czf archive.tar.gz *.txt // 将目录里所有txt文件打包成archive.tar后，并且将其用gzip压缩，生成一个gzip压缩过的包，命名为archive.tar.gz
    • tar –cjf archive.tar.bz2 *.txt // 将目录里所有txt文件打包成archive.tar后，并且将其用bzip2压缩，生成一个bzip2压缩过的包，命名为archive.tar.bz2
    • tar –cZf archive.tar.Z *.txt // 将目录里所有txt文件打包成archive.tar后，并且将其用compress压缩，生成一个umcompress压缩过的包，命名为archive.tar.Z
    • tar -rf archive.tar *.md // 增加文件到包中
    • tar -uf archive.tar other.txt // 更新包中的名为other.txt的文件
    • rar a archive.rar *.txt // rar格式的压缩，需要先下载rar for linux
    • zip archive.zip *.txt // zip格式的压缩，需要先下载zip for linux

  • 解压

    • tar –xvf archive.tar // 解压tar
    • tar -xzvf archive.tar.gz // 解压tar.gz，可用gzip -d或者gunzip解压
    • tar -xjvf archive.tar.bz2 // 解压tar.bz2，可用bzip2 -d或者用bunzip2解压
    • tar –xZvf archive.tar.Z // 解压tar.Z，可用uncompress解压
    • unrar e archive.rar // 解压rar
    • unzip archive.zip // 解压zip

  • 查看

    • tar -tf archive.tar // 列出archive.tar文件里已被打包的文件。如果加上开关v，则能列出权限、所有者、大小、时间、文件名等信息。

      为防止tar炸弹攻击，应该养成解压前查看tar文件内容的好习惯。
