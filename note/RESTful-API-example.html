resources

资源

获取资源列表

GET /resources

Parameters

NameTypeDescription
sortstringcreated 或 updated，默认 created
directionstringasc 或 desc，缺少 sort 参数将忽略

Response

  • Status: 404 Not Found
  • Status: 200 OK
  • page: 1
  • per_page: 1
{
  "total_count": 100,
  "items": [ ]
}

获取资源详情

GET /resources/:id

Parameters

NameTypeDescription
idinteger资源编号

Example

{
  "id": "资源编号"
}

Response

  • Status: 404 Not Found
  • Status: 200 OK
{
  "body": "资源内容"
}

创建资源

POST /resources

Input

NameTypeDescription
keystring提交内容(*)，Note: 待定

Example

{
  "body": "资源内容"
}

Response

  • Status: 400 Bad Request
  • Status: 201 Created
  • Location: /resources/:id
{
  "body": "资源内容"
}

编辑资源(PATCH)

PATCH /resources/:id，增删改资源的 Conditions (部分)

Input

NameTypeDescription
contentstring提交内容(*)，Note: 待定

Example

{
  "content": "资源部分内容"
}

Response

  • Status: 400 (Bad Request)
  • Status: 409 (Conflict)
  • Status: 415 (Unsupported Media Type)
  • Status: 200 OK
{
  "content": "资源部分内容"
}

编辑资源(PUT)

PUT /customers/:id/orders/:id，完整的修改资源

Input

NameTypeDescription
bodystring提交内容(*)，Note: 待定

Example

{
  "body": "资源内容"
}

Response

  • Status: 200 OK
  • Status: 201 Created， as with a POST method
  • Status: 204 (No Content)
  • Status: 409 (Conflict)
{
  "body": "资源内容"
}

删除资源

DELETE /resources/:id

Response

  • Status: 204 No Content
  • Status: 404 Not Found
