FFmpeg

  • 主要参数：

    • -i 设定输入流
    • -f 设定输出格式
    • -t hh:mm:ss[.xxx]：设定输入文件的终止时间点
    • -ss hh:mm:ss[.xxx]：设定输入文件的起始时间点

  • 视频参数：

    • -b 设定视频流量，默认为200Kbit/s
    • -r 设定帧速率，默认为25
    • -s 设定画面的宽与高
    • -aspect 设定画面的比例
    • -pix_fmt 设定视频流的图像格式（如RGB还是YUV）
    • -vcodec 设定视频编解码器，未设定时则使用与输入流相同的编解码器
    • -vn 不处理视频

  • 音频参数：

    • -ab 设定音频流的比特率，整数，单位bps。
    • -ar 设定音频流的采样率，整数，单位Hz。
    • -ac 设置音频流的声道数目。
    • -acodec 设定声音编解码器，未设定时则使用与输入流相同的编解码器
    • -an 不处理音频

  • 支持处理列表：

    • -formats：列出支持的文件格式
    • -codecs：列出支持的编解码器
    • -decoders：列出支持的解码器
    • -encoders：列出支持的编码器
    • -protocols：列出支持的协议
    • -bsfs：列出支持的比特流过滤器
    • -filters：列出支持的滤镜
    • -pix_fmts：列出支持的图像采样格式
    • -sample_fmts：列出支持的声音采样格式

INPUT: 载入文件

OUTPUT: 输出文件

转换格式

ffmpeg -i INPUT OUTPUT

eg.

ffmpeg -hide_banner -i input.flac -id3v2_version 3 out.mp3

ffmpeg -i INPUT -map 0 -c:v libx264 -c:a copy OUTPUT

裁剪视频

操作前建议用 ffprobe -hide_banner input.mp4 查看多媒体详细信息

ffmpeg -ss START -t DURATION -i INPUT -vcodec copy -acodec copy OUTPUT (几秒的误差)

eg.

ffmpeg -ss 00:00:00 -t 00:02:06.111 -i input.mp4 -vcodec copy -acodec copy output.mp4

转换前帧间编码（文件小），转换后帧内编码（文件大）。

ffmpeg -i INPUT -sameq -intra OUTPUT

ffmpeg -ss START -vsync 0 -t DURATION -i INPUT -vcodec VIDEOCODEC -acodec AUDIOCODEC OUTPUT

eg.

ffmpeg -i input.mpg -sameq -intra output.mpg

ffmpeg -ss 00:00:30 -vsync 0 -t 00:00:30 -i input.mpg -vcodec libx264-acodec libfaac output.mpg

合并视频

1. ffmpeg  -i INPUT -f mpeg  OUTPUT
2. copy -b INPUT+INPUT OUTPUT
3. ffmpeg -i INPUT -f FORMAT OUTPUT

eg.

ffmpeg.exe -i v.webm -i a.webm -acodec copy -vcodec copy muxed.webm

1.FFmpeg concat 协议

  • 视频容器是MPEG-1, MPEG-2 PS或DV等

eg.

ffmpeg -i "concat:input1.mpg|input2.mpg|input3.mpg" -c copy output.mpg

  • 对于非 MPEG 格式容器，但是是 MPEG 编码器（H.264、DivX、XviD、MPEG4、MPEG2、AAC、MP2、MP3 等），可以包装进 TS 格式的容器再合并。

    步骤：转换为mpeg/ts格式 - 合并视频 - 视频编码

eg.

ffmpeg -i input1.flv -c copy -bsf:v h264_mp4toannexb -f mpegts input1.ts

ffmpeg -i input2.flv -c copy -bsf:v h264_mp4toannexb -f mpegts input2.ts

ffmpeg -i input3.flv -c copy -bsf:v h264_mp4toannexb -f mpegts input3.ts

ffmpeg -i "concat:input1.ts|input2.ts|input3.ts" -c copy -bsf:a aac_adtstoasc -movflags +faststart output.mp4 (movflags +faststart 可边下边看)

2.FFmpeg concat 分离器

新建 filelist.txt 文件，保存要合并的视频：

file 'input-1.mkv'
file 'input-2.mkv'
file 'input-3.mkv'

ffmpeg -f concat -i filelist.txt -c copy output.mkv

3.Mencoder 连接文件并重建索引

没有使用 MPEG 编码器的视频（如 FLV1 编码器）

encoder -forceidx -of lavf -oac copy -ovc copy -o output.flv input1.flv input2.flv input3.flv

4.使用 FFmpeg concat 过滤器重新编码(有损) (复杂，跳过)

ffmpeg -i input1.mp4 -i input2.webm -i input3.avi -filter_complex '[0:0] [0:1] [1:0] [1:1] [2:0] [2:1] concat=n=3:v=1:a=1 [v] [a]' -map '[v]' -map '[a]' <编码器选项> output.mkv

[0:0] [0:1] [1:0] [1:1] [2:0] [2:1] 多个流，分别表示第一个输入文件的视频、音频、第二个输入文件的视频、音频、第三个输入文件的视频、音频。concat=n=3:v=1:a=1 表示有三个输入文件，输出一条视频流和一条音频流。[v] [a] 就是得到的视频流和音频流的名字，注意在 bash 等 shell 中需要用引号，防止通配符扩展。

视频封装

视频 + 音频 = 新的视频

ffmpeg –i INPUT_VIDEO –i INPUT_AUDIO –vcodec copy –acodec copy OUTPUT

视频与图片

1.从视频导出高质量GIF

新建文本文件，将下面代码保存，并改文件名和后缀名为 gif.bat

@echo off
setlocal enabledelayedexpansion

set gif_filename="%2.gif"
set start_time=%3
set duration=%4

set palette="palette.png"

set filters="fps=%5,scale=%6:-1:flags=lanczos"

ffmpeg -v warning -ss %start_time% -t %duration% -i %1 -vf "%filters%,palettegen=max_colors=%7:stats_mode=diff" -y %palette%
ffmpeg -v warning -ss %start_time% -t %duration% -i %1 -i %palette% -lavfi "%filters% [x]; [x][1:v] paletteuse=dither=floyd_steinberg" -y %gif_filename%

将 gif.bat 放在视频所在文件夹，

在视频文件夹图标上，按住 Shift， 点击鼠标右键选择：在此处打开命令窗口

运行命令：

gif.bat input.mp4 output 00:00:25 00:00:35 10 640 256

对应解释：

gif.bat 导入视频源 导出Gif图片名 开始时间 结束/持续时间 帧率 宽度 色彩数(4-256)

2.整个视频导出高质量GIF

gif-all.bat

@echo off
setlocal enabledelayedexpansion

set gif_filename="%2.gif"

set palette="palette.png"

set filters="fps=%3,scale=%4:-1:flags=lanczos"

ffmpeg -v warning -i %1 -vf "%filters%,palettegen=max_colors=%5:stats_mode=diff" -y %palette%
ffmpeg -v warning -i %1 -i %palette% -lavfi "%filters% [x]; [x][1:v] paletteuse=dither=floyd_steinberg" -y %gif_filename%

调用：gif-all.bat input.mp4 output 10 640 256

FPS

  • 5 (max 60 seconds)
  • 7 (max 40 seconds)
  • 10 (max 30 seconds)
  • 20 (max 15 seconds)
  • 25 (max 10 seconds)

3.联合 ImageMagick's 使用(复杂，跳过)

eg.

ffmpeg -ss 00:00:06 -t 5 -i input.mp4 -vf scale=320:-1 -r 25 -f image2pipe -vcodec ppm - | convert -delay 5 -loop 0 - gif:- | convert -layers Optimize - 3.gif

ffmpeg -ss 00:00:06 -t 5 -i input.mp4 -vf scale=640:-1:flags=lanczos,fps=10 frames/ffout%03d.png

convert -loop 0 frames/ffout*.png output.gif

视频导出图片

eg.

ffmpeg -i input.mp4 -r 1 -s W*H/640*320 -f image2 foo-%03d.jpeg

图片合并为视频

eg.

ffmpeg -f image2 -framerate 12 -i foo-%03d.jpeg -s W*H input.mp4

ffmpeg -f image2 -pattern_type glob -framerate 12 -i 'foo-*.jpeg' -s W*H input.mp4

4.导出图片

eg.
ffmpeg -i input.mp4 output%d.jpg

ffmpeg -i input.mp4 -r 25 output%d.jpg (-r:帧率,每秒截取图片数量,默认25)

ffmpeg -i input.mp4 -r 25 -ss 00:00:00 -t 00:00:01 output%02d.jpg

ffmpeg -y -i input.mp4 -ss 00:00:22.000 -vframes 1 -an output.jpg

ffmpeg -i input.mov -ss 00:00:06.060 -t 5 -pix_fmt rgb24 -r 10 -vf scale=640:-1 output.gif

ffmpeg -ss 0 -t 10 -i input.mp4 -f gif output.gif

ffmpeg -ss 0 -t 10 -i input.mp4 -s 320x240 -f gif output.gif

ffmpeg -ss 0 -t 10 -i input.mp4 -s 320x240 -f gif -r 1 output.gif

ffmpeg -ss 0 -t 10 -i input.mp4 -r 1 -s 320x240 -f image2 output-%03d.jpeg

ffmpeg -f image2 -framerate 5 -i output-%03d.jpeg output.gif (合成帧率:5fps)

分离视频流(音频/视频)

ffmpeg -i INPUT -vcodec copy -an OUTPUT_VIDEO
ffmpeg -i INPUT -acodec copy -vn OUTPUT_AUDIO

eg.

ffmpeg –i input.mp4 –vcodec copy –an –f m4v output.264

ffmpeg -i input.mp4 -f mp3 audio.mp3

ffmpeg -i input.mp4 -vn audio.mp3

在输入文件中有多个音频流，这些流由“0：0”和“0：1”标识。 用-map来选择输出流。

ffmpeg -i INPUT -map 0:1 out.wav

将比较特殊的UYVY格式的YUV文件转换成无损压缩的avi格式 (监控视频)

eg.

ffmpeg -s 1280×1024 -f rawvideo -pix_fmt uyvy422 -i input.yuv -vcodec huffyuv|rawvideo output.avi

加水印

Top/Right/Bottom/Left

TR: ffmpeg -i INPUT -i watermarklogo.png -filter_complex overlay="main_w-overlay_w-10:10" OUTPUT
TL: ffmpeg -i INPUT -i watermarklogo.png -filter_complex overlay="10:10" OUTPUT
BL: ffmpeg -i INPUT -i watermarklogo.png -filter_complex overlay="10:main_h-overlay_h-10" OUTPUT
BR: ffmpeg -i INPUT -i watermarklogo.png -filter_complex overlay="main_w-overlay_w-10:main_h-overlay_h-10" OUTPUT

视频过渡

按帧：

fade=in:0:50 = fade=t=in:s=0:n=50 = fade=type=in:start_frame=0:nb_frames=50

按时：

fade=in:st=0:d=2 = fade=in:start_time=0:duration=2

eg.

ffmpeg -i input.mp4 -ss 20 -t 20 -vf "fade=in:0:50,fade=out:450:50:c=black" output.mp4

ffmpeg -i input.mp4 -ss 20 -t 20 -vf "fade=in:0:d=2,fade=out:st=29:d=1:c=black" output.mp4

ffmpeg -i input.mp4 -vf "fade=out:st=00:00:31.94:d=3:c=black" output.mp4

ffmpeg -i input.mp4 -vf "fade=in:st=0:d=2:c=black,fade=out:st=00:00:36.07:d=3:c=black" output.mp4

音频过渡

ffmpeg -i input.mp3 -af "afade=enable='between(t,0,3)':t=in:ss=0:d=3,afade=enable='between(t,27,30)':t=out:st=27:d=3" -t 30 output.mp3

ffmpeg -i input.mp3 -af "afade=t=in:st=0:d=2,afade=t=out:st=27:d=3" output3.mp3

最后渐出

ffmpeg -i input.mp4 -ss 0 -t 10 -vf "fade=out:st=8:d=2" -af "afade=t=out:st=8:d=2" output.mp4

开始渐入，最后渐出

ffmpeg -i input.mp4 -vf "fade=in:st=0:d=1,fade=out:st=6:d=1" -af "afade=t=in:st=0:d=1,afade=t=out:st=6:d=1" output.mp4
