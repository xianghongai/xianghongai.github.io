加密的网络传输协议，在不安全的网络中为网络服务提供安全的传输环境.

  • Open .ssh Directory
    $ cd ~/.ssh/

  • Checking for existing SSH keys
    $ ls -al ~/.ssh

  • Generating a new SSH key
    $ ssh-keygen -t rsa -b 4096 -C "your@email.com"

  • start the ssh-agent in the background
    $ eval $(ssh-agent -s) or eval "$(ssh-agent -s)"

  • Adding your SSH key to the ssh-agent
    $ ssh-add ~/.ssh/your_rsa

  • Add the SSH key to your GitHub account.
    $ clip < your_rsa.pub
    git.hosting - settings - SSH and GPG keys - New SSH key

  • Testing your SSH connection
    $ ssh -T git@github.com or ssh -vT git@github.com/ssh -T git@gitlab.com/ssh -T git@bitbucket.org

  • Changing a passphrase
    $ ssh-keygen -f id_rsa -p or $ ssh-keygen -p
