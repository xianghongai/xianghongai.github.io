Windows 操作技巧

Windows 7 以 上

  • Windows+上下左右，当前窗口 最大化、左侧放置、右侧放置、最小化(Windows+0)
  • Windows+Shift+左右，窗口在多个屏幕之间移动
  • Windows+Space，临时看一下桌面，放开恢复
  • Windows+加/减，放大镜放大/缩小
  • Windows+数字键，按顺序打开相应程序，与快速启动栏中程序相对应（索引从1开始）
  • Windows+T，切换不同的任务栏按钮（通过设置，同一应用的窗口会合并成一个任务栏按钮）
  • Windows+P，切换多显示桌面扩展方式
  • 拖动窗口摇动，其他窗口最小化，再摇一摇其它窗口恢复显示
  • 文件夹空白处按住Shift，右键弹出菜单，可以看到“在此处打开命令行窗口”
  • Ctrl+Shift+N，新建文件夹
  • Windows+TAB（后退是 Windows+Shift+Tab），切换窗口是有3D桌面效果的（需开启Aero特效）

Tab重命名用法：

  • 文件F2重命名，按Tab键，进入下一个文件重命名状态
  • 按Shift+Tab，进入上一个文件重命名状态。

Windows 相关快捷键

  • Windows+D，显示桌面
  • Windows+L，锁屏
  • Windows+E，打开资源管理器
  • Windows+R，打开运行对话框
  • Windows+Pause，显示系统属性对话框
  • PrintScreen，截屏整个桌面到剪切板
  • Alt+PrintScreen，截屏当前窗口到剪切板
  • Alt+F4，关闭应用程序
  • Ctrl+Esc，打开开始菜单
  • Ctrl+Shift+Esc，打开任务管理器
  • Alt+Tab，正向切换当前活动程序，反向（Alt+Shift+Tab）
  • Ctrl+Shift，输入法切换
  • Ctrl+空格，中英文切换

资源管理器相关快捷键

  • F2，重命名
  • F5，刷新当前窗口
  • Ctrl+A，全选（其他多选方式：1.按住Ctrl，点选； 2.按住Shift，点选）
  • Ctrl+Z，撤销；Ctrl+C，拷贝；Ctrl+V，粘贴；Ctrl+X，剪切
  • Alt + 菜单字母（文件(F)），焦点定位
  • Alt+D，定位到地址栏
  • Alt+空格，打开系统菜单
  • Alt+Enter，打开所选文件或目录的属性对话框
  • Alt+向上，返回上一层文件夹
  • BackSpace，返回前一步
  • Shift+F10，弹出右键菜单
  • Shift+Delete，永久删除文件或文件夹
